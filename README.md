# Relógio em JavaScript

Projeto desenvolvido para praticar JavaScript e manipulação do DOM, exibindo o horário em tempo real de forma dinâmica.

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Exibição do horário atual
- Atualização automática em tempo real
- Manipulação de elementos do DOM

## Objetivo do projeto
Consolidar conhecimentos em JavaScript, lógica de programação e interação com a interface.

## Como executar
- Clone o repositório
- Abra o arquivo index.html no navegador
